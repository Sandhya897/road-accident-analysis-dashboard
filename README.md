# Road Accident Analysis Dashboard

## Overview
This Power BI dashboard analyzes road accident data, highlighting trends and insights to improve road safety.  

## Features
- Accident trends by year and location
- Heatmaps for high-risk areas
- Accident type and severity analysis
- Interactive filters
